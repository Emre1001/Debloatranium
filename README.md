## 🌍 About the Project
**Debloatranium** is a premium, open-source system utility designed to transform your Windows experience from a cluttered, slow environment into a high-performance, minimalist workstation. Using a high-end Glassmorphism UI, it provides even non-technical users with the power of advanced registry and service optimizations.

## ✨ Key Features
- **💎 Multi-Billion Company Aesthetics**: A stunning, semi-transparent liquid UI with fluid animations.
- **🛡️ Multi-Language Support**: Full native support for **English (EN)**, **German (DE)**, and **Turkish (TR)**.
- **🚀 Ultra-Construction Mode**: One-click cleanup that enables Dark Mode, fixes Windows 11 Taskbar alignment, and purges AI bloat (Copilot/Recall).
- **🕹️ Modus Slider**: From "Minimal" to "Zero ☢️" – pick your level of performance.
- **🛠️ Expert Mode**: Toggle every single mini-setting manually for total control.
- **📦 Portable**: Single `.exe` execution – no installation required.

## 🚀 Getting Started
### Quick Start (Portable)
1. Download the latest `Debloatranium.exe` from the [Releases](https://github.com/Emre1001/Debloatranium/releases) page.
2. Run as **Administrator**.
3. Choose your language and start constructing your perfect OS.

### For Developers
```bash
git clone https://github.com/Emre1001/Debloatranium.git
cd Debloatranium/App
npm install
npm start
```
